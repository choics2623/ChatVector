# ChatVector
### How to Use
```
CHAT_PATH="adress"
PT_PATH="adress"
CPT_PATH="adress"
SAVE_PATH="adress"
UPLOAD_PATH="adress"
python chat_vecter.py \
     --chat_path "$CHAT_PATH" \
     --pt_path "$CPT_PATH" \
     --cpt_path "$CPT_PATH" \
     --save_path "$SAVE_PATH" \
     --upload_path "$UPLOAD_PATH" \
     --lora
```
